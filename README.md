# Crown Heights Academy Website

A modern, responsive website for Crown Heights Academy, Woji, Port Harcourt.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- JavaScript (ES6+)

## Structure
- `/css` - Stylesheets
- `/js` - Logic and interactions
- `/images` - Static assets
- `/assets` - Downloadables (PDFs etc)

## Setup
Simply open `index.html` in your browser.

## Deployment
1. **Netlify**: Drag and drop the root folder to Netlify Drop.
2. **GitHub Pages**: Push to a repository and enable Pages from Settings > Pages > Source (main branch).
